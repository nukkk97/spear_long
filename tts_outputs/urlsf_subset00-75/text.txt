Back to home May Run ansible playbook on TripleO nodes by Juan Antonio Osorio Robles Running an ansible playbook on TripleO nodes is fairly simple thanks to the work done by the folks working on tripleo validations Theres no need to manually maintain an inventory file with all the nodes as there is already dynamic inventory script set up for us So using an ansible playbook would look as the following source stackrc ansible playbook usr bin tripleo ansible inventory path to playbook yaml This will use localhost for the undercloud node and fetch the nodes from nova to get the overcloud nodes There are also roles already available such as controllers and computes which can be accessed in your playbooks with the keys controller and compute respectively And support is coming for dynamic roles as well So for simple example lets say we want to install the Libreswan package in all the overcloud nodes It would be done with playbook that looks as the following hosts overcloud become true tasks name Install libreswan yum name libreswan state latest tags tripleo openstack Back to home txt ustar Donald Trump and Vladimir Putin Thomson Reuters President Donald Trump and the Kremlin offered differing accounts of meeting Trump
